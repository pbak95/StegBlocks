//Instrukcja uruchomienia programu do weryfikacji plików pcap:
1. Plik wykonywalny PcapVerify.jar

2. Na komputerze musi byc zainstalowana java

3. pliki pcap musza byc w tym samym folderze co plik wykonywalny jar

4. aby zweryfikowac plik nalezy uzyc komendy:
	java -jar PcapVerify.jar pcap_file_name.pcap

5. Wynik programu to tekst oznajmiajacy czy dany plik pcap zawiera czy nie zawiera tajnych danych
